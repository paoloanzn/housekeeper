# housekeeper

Minimal Bun CLI (TypeScript).

## Run

```bash
bun run index.ts --help
# or
bun start --help
```

## Development

```bash
bun dev
```

## Build standalone binary

```bash
bun run build
./dist/housekeeper --help
```

## Install globally (dev)

```bash
bun link
housekeeper --version
```

## Commands

```
housekeeper hello [name]
housekeeper --version
housekeeper --help
```

## Project

Created with Bun v1.3.11. Zero runtime dependencies.
